# Welcome to Hangman

## How to begin

1. Open your terminal and type node .
2. Press CTRL + C to stop.

## Rules

1. You have 6 tries to guess a letter in the randomly selected word. The round will end when the player has used up all of their guesses or has guessed the correct word.
2. The game will keep track of the number of rounds played, as well as wins and losses.
